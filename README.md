# Ödev 4: Modern kart tasarımı

<img width="1326" height="869" alt="image" src="https://github.com/user-attachments/assets/cbfef67b-2aab-4fb5-b3cc-1178b27470d8" />

* Görselde verilen kart tasarımı yapılacak.
* Bir html dosyası ve bir external css dosyası oluşturulacak.
* Html dosyası temel html yapısına sahip olmalı.
* `body`'ye `linear-gradient` uygulanacak, gerekli miktarda boşluk bırakılacak.
* Google fonts'tan bir font eklenecek.
* Global stil resetlenecek.
* Her kartın html yapısı aşağıdaki gibi olacak.
* <img width="592" height="305" alt="image" src="https://github.com/user-attachments/assets/8c3c13c6-62bc-469f-9b8f-006e9d0d0fb9" />
* CSS dosyası kart tasarımı baz alınarak yapılacak.
* Her üç kart bir tane `<div class="cards-container">...</div>`'in içine yerleştirilebilir.
* Bu `div`'e ait stil: `.cards-container {
            display: flex;
            gap: 30px;
            justify-content: center;
        }
        `
* Yaptığınız web sayfası yayınlanacak.
